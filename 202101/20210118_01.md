# 元字符

匹配内容的规则

+ [] [^]
+ \w  数字字母下划线
+ \d   只匹配数字
+ \n
+ \t
+ \s   空白
+ \S   非空白
+ \w  非数字字母下划线
+ \D  非数字
+ [\d\D] [\w\W] [\s\S]  匹配所有
+ .匹配除换行符之外的所有
+ ^ 匹配一个字符串的开始
+ $ 匹配一个字符串的结尾
+ a表达式|b表达式  匹配a表达式或b表达式，如果匹配a成功了，不会继续匹配b，所以如果有重叠的部分，总是把长的放在前面
+ \  转义
+ ()  约束|描述的内容的范围问题
+ \b

# 量词

+ {n}  表示匹配n次
+ {n,}  表示至少匹配n次
+ {n,m} 表示至少匹配n次，至多m次
+ ？  表示匹配0次或1次
+ `+` 表示匹配1次或多次
+ `*` 表示匹配0次或多次
